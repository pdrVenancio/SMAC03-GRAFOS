Tipos de grafos:
	Grafo simples: padrão, aresta via dupla sem peso, sem laço(loop).
	Grafo dirigido(digrafo): arestas direcionadas.
	Multigrafo: arestas paralelas.
	Pseudografo: arestas paralelas e laços.
	Grafo valorado: as arestas possuem pesos.
	Grafo completo: notação Kn (onde n é o numero de arestas).
	Subgrafo: subparte de um grafo existente.
	Grafo regular: quando todos os seus vértices tem o mesmo grau.
	Grafo completo: É o grafo que somado a grafo inicial o torna um grafo completo, todos os vértices são interligados.
	Grafo bipartido: Onde um grupo de vertice so se comunica com vertices de outro grupo. Ele será bipartido se não possuir nenhum ciclo de tamanho ímpar (nesse ciclo não podemos voltar arestas).
	Grau de um vertice: somamos as conexões de cada vertice do grafo, para grafos dirigitos somamos tanto a entrada como saida( enclusive me loops).
	Densidade de um grafo: relaçao de arestas e vertices de um grafo que pode variar de 0 a 1, exceto em pseudografos e multgrafos onde ela pode passar de 1 por termos que contar aresta multiplas e loops.
	Componente conexo: conectividade de todos os seus vertices por arestas.

Isomorfismo: Uma representação diferente para o mesmo grafo. Para provar que dois grafos simples são isomorfos é necessário encontrar uma bijeção(função que para cada elemento do conjunto A tem um representante no conjuto B) e mostrar que a propriedade da adjacência é preservada. 

Planaridade: grafo ou pseudografo, direcionados ou não, desenhado em uma superfície plana de tal modo que suas arestas não se cruzam. Se um grafo possuir gruzamentos, mas é possivel ajustar para nao possuir cruzamentos dizemos ser um grafo planar.

Passeio: É permitido repetir vertice e aresta.
	para grafos simples apenas apresnetamos os vertices para representar.
	para multgrafos e pseudografos representamos tanto os vertices como as arestas.
	para grafos dirigidos apenas podemos seguir o caminho das setas.
	psseio aberto vertice inicial e final são diferentes, passeio fechado eles vão ser o mesmo vertice.

Trajeto: É permitido repetir vértices mas não é permitido repetir arestas.
	os outros conseitos sao iguais ao do passeio.

Caminho: Não é permitido repetir vértices e arestas.
	os outros conseitos sao iguais ao do passeio.

Ciclo: Segue os conceitos do caminho fechado, ou seja o vertice inicial e final são os mesmos e não repetimos vertices e arestas.

DAG (Directed Acyclic Graph): é um grafo direcionado onde nao é possil formarmos um ciclo.

Fecho transitivo direto: conjunto de todos os vertices que podem ser alcansados a partir de uma vertice inicial.
Fecho transitivo indireto: o conjunto de todos os vértices de um grafo a partir dos quais v é alcançado.

Caminho Euleriano: é um caminho em um grafo que visita toda aresta exatamente uma vez, tendo que passar ao menos uma vez em cada vertice.
Caminho Hamiltoniano: é um caminho que passa exatamente uma vez em cada vertice. Caso ele retorne ao verice inicial(ciclo hamiltoniano) é dito um grafo hamiltoniano, caso contrario (caminho hamiltoniano) é dito um grafo semi-hamiltononiano.

Conectividade: Conectividade está relacionada a um número mínimo de elementos (vértices ou arestas) que precisam ser removidos para desconectar os vértices restantes uns dos outros.	
	Grafo não direcionado(GND): conexo, todos os vértices são alcançáveis a partir de qualquer outro.
	Grafo Desconexo: (grafos possuem partes disconectadas) O grafo possui duas componentes não existindo uma ligação entre elas. 
	Grafo simplesmente conexo (s-conexo): Se G é um grafo direcionado então ele é considerado conexo quando o seu grafo não-direcionado subjacente é conexo.
	Grafo Semi-Fortemente Conexo (sf-conexo): Para um grafo direcionado, se cada par de vertice exite uma caminho entre eles podendo ser de vi para vj ou vj para vi.
	Grafo Fortemente Conexo (f-conexo): Para um grafo direcionado, se cada par de vertice exite uma caminho entre eles podendo ser de vi para vj e vj para vi.
	Componentes Fortemente Conexos: Se G é um grafo direcionado, componentes fortemente conexos são subgrafos maximais fconexos (todos os vértices de um mesmo componente são alcançáveis entre si).

Conectividade em vertice: conectividade em vértices k(G) de um grafo G corresponde a menor quantidade de vértices cuja remoção torna G desconexo.

k-Conectividade: existirem mais de um caminho sem repetir aresta que ligue dois vertices.

Articulação: 
	Vertice de articulação:	é aquele cuja remoção resulta na desconexão de G.
	Aresta de articulação: é aquela cuja remoção resulta na desconexão de G.

Formulas: 
	Total de arestas de um grafo completo é dada por: modulo V ao quadrado menos modulo V sobre 2.
	Para grau de grafos nao direiconados: 2 vezes arestas.
	Densidade grafo simples: 2 vezes aresta sobre vertice menos 1 vezes vertice.
	Densidade digrafo: aresta sobre vertice menos 1 vezes vertice.
	Planaridade Numero de regioes: aresta menos vertice mais 2.
	Planaridade provar que o grafo é planar: aresta menor igual 3 vezes vertice menos 6.
	Planaridade provar que grafo bipartido: aresta menor igual duas vezes vertice menos 4.
